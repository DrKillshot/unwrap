# @unnullable/unwrap

Unwrap exposes functional programming monads with a syntactic sugar to help create more expressive and declarative web applications.

## What's to unwrap?
- `Maybe<Value>`
- `Either<Ok, Error>`
- `IO<Operation>`

## Installation
```sh
npm install @unnullable/unwrap

# ------ or ------------------

yarn add @unnullable/unwrap
```
